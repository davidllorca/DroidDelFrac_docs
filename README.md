# DroidDelFrac [Good memories]

>Final app project of my student times :grin:

App for controlling and sharing expenses of a event. Event's creator could assign and distribute expenses to members.


## Technology
- Eclipse
- Java
- Android
- SQLite
- API Facebook
- API Google+
- API PayPal
- API Google Wallet
